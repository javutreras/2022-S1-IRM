Propiedades de las relaciones binarias

Definición. Sea RRR una relación binaria entre elementos de AAA y BBB. Se definen los conjuntos imagen directa de a∈Aa\in Aa∈A: R(a)={x : (a,x)∈R}R(a)=\{x\,:\,(a,x)\in R\}R(a)={x:(a,x)∈R}
e imagen inversa de b∈Bb\in Bb∈B: R−1(b)={x : (x,b)∈R}R^{-1}(b)=\{x\,:\,(x,b)\in R\}R−1(b)={x:(x,b)∈R}

Definición. La misma definición anterior se puede realizar a subconjuntos. Se definen la imagen directa de A0⊆AA_0\subseteq AA0​⊆A: R(A0)=⋃a∈A0R(a)R(A_0)=\bigcup_{a\in A_0}R(a)R(A0​)=a∈A0​⋃​R(a)
y la imagen inversa de B0⊆BB_0\subseteq BB0​⊆B: R−1(B0)=⋃b∈B0R−1(b)R^{-1}(B_0)=\bigcup_{b\in B_0}R^{-1}(b)R−1(B0​)=b∈B0​⋃​R−1(b)

Definición. Sea RRR una relación binaria entre elementos de AAA y BBB. Si para cada a∈Aa\in Aa∈A el conjunto imagen directa R(a)R(a)R(a) tiene exactamente un elemento, decimos que RRR es una función R:A→BR:A\to BR:A→B, y a ese único elemento lo llamamos la imagen de aaa bajo RRR, y lo denotamos también como R(a)R(a)R(a).

Notación. Si RRR es una relación binaria entre un conjunto AAA y un conjunto BBB, el conjunto AAA se llama el dominio de RRR, y el conjunto BBB el codominio de RRR.

Notación. Una relación binaria sobre un conjunto AAA es una relación cuyo dominio y codominio son ambos el conjunto AAA.

  • R2={(x,y) : x+y=0}R_2 = \{(x,y)\,:\,x+y=0\}R2​={(x,y):x+y=0}

  • R3={(x,y) : ∣x∣=∣y∣}R_3=\{(x,y)\,:\,|x|=|y|\}R3​={(x,y):∣x∣=∣y∣}

  • R4={(x,y) : y=x2}R_4=\{(x,y)\,:\,y=x^2\}R4​={(x,y):y=x2}

Clasificación de las relaciones binarias sobre un conjunto

Definiciones. Sea RRR una relación sobre un conjunto AAA.

  • Se dice que RRR es reflexiva si para todo x∈Ax\in Ax∈A se tiene (x,x)∈R(x,x)\in R(x,x)∈R.

  • Se dice que RRR es simétrica si cada vez que (x,y)∈R(x,y)\in R(x,y)∈R se tiene (y,x)∈R(y,x)\in R(y,x)∈R.

  • Se dice que RRR es antisimétrica si cada vez que (x,y)∈R ∧ (y,x)∈R(x,y)\in R\,\wedge\,(y,x)\in R(x,y)∈R∧(y,x)∈R se tiene x=yx=yx=y.

  • Se dice que RRR es transitiva si cada vez que (x,y)∈R ∧ (y,z)∈R(x,y)\in R\,\wedge\, (y,z)\in R(x,y)∈R∧(y,z)∈R se tiene (x,z)∈R(x,z)\in R(x,z)∈R.

Definiciones.

  • Una relación es de equivalencia si es reflexiva, simétrica y transitiva.

    • Si RRR es una relación de equivalencia y (a,b)∈R(a,b)\in R(a,b)∈R, se dice que aaa y bbb son equivalentes. Se escribe a≡Rba\equiv_R ba≡R​b.

  • Una relación es de orden si es reflexiva, antisimétrica y transitiva.

    • Si RRR es una relación de orden y (a,b)∈R(a,b)\in R(a,b)∈R, se dice que aaa es menor o igual que bbb, según RRR. Se escribe a≤Rba\le_R ba≤R​b.

Ejemplos.

  • La relación dejar el mismo resto al dividir por 222 es una relación de equivalencia en N+\mathbb{N}^+N+.

  • La relación dividir a es una relación de orden en N+\mathbb{N}^+N+.
